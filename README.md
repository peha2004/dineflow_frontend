# 🍽️ DineFlow - Restaurant Reservation System

DineFlow is a full-stack restaurant reservation and management system built using React, TypeScript, Vite, Node.js, Express.js, and MongoDB.

The system allows customers to create reservations, select tables, order menu items, generate QR codes, and manage their bookings. Administrators can manage users, tables, menu items, and monitor reservation history through a dedicated dashboard.

## Features

### Customer Features
- User Registration & Login
- JWT Authentication
- Create Table Reservations
- Select Event Types (Birthday, Party, Anniversary, Corporate, Normal Dining)
- View Restaurant Menu
- QR Code Generation for Reservations
- Reservation History Management
- User Profile Management

### Admin Features
- Dashboard Analytics
- User Management
- Table Management
- Menu Management
- Reservation Monitoring
- View Customer Booking Details
- View Special Requests

## Technologies Used
